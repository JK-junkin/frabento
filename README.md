`frabento` provides assets for FRA's job using R language.

## インストール

このページ (リポジトリ) から`frabento`パッケージの開発およびリリースバージョンを取得できます.

``` r
# installation
install.packages("devtools")
devtools::install_github("JK-junkin/frabento@dev")

# load package
library(frabento)
```
